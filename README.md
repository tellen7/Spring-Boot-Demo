## Spring Boot进阶学习Demo
**技术栈**
Spring Boot + Spring JPA + Spring AOP + Hibernate + mysql

**技术实现**
1. profiles环境配置，实践了开发环境和生产环境
2. 表单`@Valid`验证
3. Spring AOP技术用于请求日志记录
4. 统一异常处理`@ControllerAdvice`,`@ExceptionHandler`处理自定义异常和未知异常
5. 单元测试，测试service方法 和 controller路径请求测试
6. JPA事务处理
7. 返回数据结果统一封装
